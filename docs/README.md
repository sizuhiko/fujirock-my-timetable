# フジロック マイタイムテーブル

このドキュメントは フジロック マイタイムテーブル Web アプリケーションの仕様書です。

このアプリケーションには大きく分けて3つの機能があります

- [マイタイムテーブル閲覧](./viewer.md)
- [マイタイムテーブル編集](./editor.md)
- [マイテント](./tent.md)

それぞれの機能は、リンク先のドキュメントを参照してください。

## 用語整理

### マイタイムテーブル

フジロック3日間の日付ごとに自分が見るステージ/アーティストだけが表示されるタイムテーブル。
一般的なフェスのマイタイムテーブルと異なるのは、ステージごとの空き時間にフードエリアに行く予定も登録できるのが特徴。

### マイテント

自分がテントをはった場所（位置）を記録できる。
ステージ会場からテントに戻るとき、自分のテントまで Google Map で徒歩ナビゲーションで戻るために利用する。

## UI の改善メモ

ここはアプリの全体のUIに関する改善メモを記録します。

- nav エリアの フジロック マイタイムテーブル というタイトルと、ページを移動できるメニューがあるので、以下のようにしてください
  - nav エリアの ` フジロック マイタイムテーブル` を h1 のタイトルとする。スタイルは現時点のままでok
  - main エリアにある h1 と遷移するボタンは不要
- nav エリアは縦スクロールしても常に表示されるようにする
