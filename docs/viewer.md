# マイタイムテーブル閲覧

以下の機能を持つものとします。

- ブラウザを実行した端末に保存されているマイタイムテーブル情報から、日付ごとにタイムテーブル表示できる
- 見るライブ枠は、ステージごとに色分けされていること。色は公式サイトのタイムテーブルの色と同じものを使うこと
- 見るライブ枠には、開始時間、終了時間、アーティスト名が表示されていること
- タイムテーブルの見るライブをクリックすると、フジロック公式サイトのアーティストページへ新しいタブで遷移すること
- タイムテーブルの行くフードエリアをクリックすると、フジロック公式サイトの選択したフードエリアページへ新しいタブで遷移すること
- 編集ボタンからマイタイムテーブル編集ページへ遷移できる
