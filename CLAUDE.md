# CLAUDE.md

## プロジェクト名

フジロック マイタイムテーブル

## Primary Goal（最重要目的）

フジロック当日までに計画をたててマイタイムテーブルを作成し、当日スムーズに移動を行う

## Tech Stack

- フレームワーク: Next.js
- プログラミング言語: TypeScript
- Node.js バージョン: 24以上
- コードフォーマット: Prettier
- 静的チェック: ESLint
- デプロイ: スタンドアローンビルドした結果を GitHub Pages で公開して利用する
- データの保存: ブラウザのローカルストレージやIndexedDBに保存してあれば大丈夫です。サーバーにデータベースは不要です
- ユーザー認証は不要です
- 多言語対応: 不要。日本語だけでok

## Target Platform

- Webブラウザから利用する

このアプリケーションの仕様書は docs/README.md を参照
