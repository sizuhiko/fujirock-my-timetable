# マイテント

以下の機能を持つものとします。

- マイテント位置が登録されていない場合は、追加アイコンボタンが表示されること
- 追加アイコンボタンをクリックすると、現在位置を保存すること
- マイテント位置が登録されている場合は、登録位置を中心に地図が表示され、登録位置がピン表示になっていること
- 登録位置を削除して、再登録できること
- マイテント位置が登録されている場合は、ナビゲーションボタンを押すと GoogleMap アプリが起動して徒歩モードで登録位置まで案内されること

## 実装メモ

- 地図表示には Leaflet + OpenStreetMap を使用する(Google Maps JavaScript API は API キー発行・Google Cloud 課金設定が必要なため採用しない)
- ナビゲーションボタンは Google Maps の URL スキーム(`https://www.google.com/maps/dir/?api=1&destination=...&travelmode=walking`)でアプリ/サイトを起動する方式とし、API キーは不要とする
